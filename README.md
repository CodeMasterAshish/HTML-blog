# 🚗 GearVize - Cars Under ₹6 Lakhs in India

A simple HTML-only blog website showcasing some of the best cars available in India under ₹6 Lakhs. This project focuses on semantic HTML structure and presents car information, images, pricing, mileage, and comparisons in a clean blog format.

## 🌟 Features

* Pure HTML (No CSS or JavaScript)
* Semantic HTML5 Tags
* Car Images and Descriptions
* Comparison Table
* Blog-Style Layout
* Beginner-Friendly Project
* Responsive Foundation for Future Enhancements

## 📸 Included Cars

* Maruti Suzuki Alto K10
* Maruti Suzuki S-Presso
* Renault Kwid
* Maruti Suzuki Celerio
* Maruti Suzuki WagonR

## 🚀 Getting Started

1. Clone the repository:

```bash

```

2. Open the project folder.

3. Open `index.html` in your browser.

OR

4. Use the VS Code Live Server extension for a better development experience.
